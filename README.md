Скрипт для переноса файлов с личного Яндекс.Диска на Общий диск, использует REST API Яндекса. Он рекурсивно получает список всех файлов и папок из корня личного диска, создает структуру папок на общем диске (если нужно) и перемещает ресурсы. Требуется OAuth-токен и hash общего диска (пример "vd:<здесь id>", получить из веб-интерфейса Яндекс.Диска).
​

**Требования**
Python 3.x с библиотеками requests, python-dotenv (установите зависимости: pip install -r requirements.txt).

OAuth-токен: https://oauth.yandex.ru/authorize?response_type=token&client_id=your_app_id.

Hash виртуального диска (vd:<hash>): из URL общего диска, например vd:abc123.

​**Настройка**
1. Скопируйте `env_example.txt` в `.env`
2. Заполните значения в `.env` файле:
   - `YANDEX_OAUTH_TOKEN` - ваш OAuth токен
   - `VIRTUAL_DISK_HASH` - hash общего диска

​**Использование**
Запустите: python move_to_shared_ydisk.py.

Скрипт создаст структуру папок и переместит файлы. Асинхронные операции ждут завершения.
​

**Ограничения**
Рекурсивный листинг работает только на 1 уровень (_embedded.items); для глубоких структур доработайте рекурсию.

Лимит API: 1000 элементов/запрос, пагинация обрабатывается.

Общий диск должен быть создан заранее в Яндекс.Диске.
